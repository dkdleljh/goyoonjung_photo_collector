# 고윤정 사진봇 사용설명서 (초보자용)

`goyoonjung_photo_collector`는 공개적으로 접근 가능한 소스에서 배우 **고윤정** 관련 이미지를 수집해
로컬 폴더에 저장하는 수집기입니다.

- 유료 API 강제 없음
- 원본 바이트 그대로 저장(리사이즈/재인코딩 없음)
- 중복 제거(sha256 + SQLite)
- 720p(짧은 변 720px) 미만은 기본적으로 저장하지 않음

> ⚠️ 원칙
> - 로그인/차단 우회, 비공개 콘텐츠 접근을 목표로 하지 않습니다.

---

## 0) 폴더 위치

- `~/Desktop/goyoonjung_photo_collector`

---

## 1) 설치(처음 1회)

```bash
cd ~/Desktop/goyoonjung_photo_collector
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

---

## 2) .env 설정(선택)

```bash
cp .env.example .env
nano .env
```

- 네이버 이미지 검색 OpenAPI를 쓰려면:
  - `NAVER_CLIENT_ID`
  - `NAVER_CLIENT_SECRET`
- 저장 경로를 지정하려면:
  - `PHOTO_ROOT=/원하는/경로`

> `.env`는 절대 커밋하면 안 됩니다.

---

## 3) 가장 먼저 해볼 것: 드라이런

다운로드 없이 “후보 수집만” 점검합니다.

```bash
python -m app.cli run --once --dry-run
```

---

## 4) 실제 다운로드 실행

```bash
python -m app.cli run --once
```

---

## 5) 상태 확인

```bash
python -m app.cli status
```

- 최근 실행 시간/성공(OK) 개수/종료코드/실패 사유를 봅니다.

---

## 6) 저장 위치

기본 저장 구조(예시):

```text
Desktop/Goyoonjung_Photos/
  YYYY-MM-DD/
    naver/
    wikimedia/
    instagram_seed/
    google/
  meta/
    items.jsonl
    failed.jsonl
    dedup.sqlite
    status.json
  logs/
    summary_YYYY-MM-DD.txt
```

---

## 7) Instagram seed 추가(선택)

`seeds/instagram_urls.txt`에 URL을 한 줄에 하나씩 추가합니다.

---

## 8) 자주 묻는 질문

### Q1. OK가 0인데 정상인가요?
- 드라이런이면 정상입니다(저장을 안 하니까요).
- 실제 실행에서도 중복/해상도 필터로 OK가 0일 수 있습니다.

### Q2. RESOLUTION_TOO_SMALL이 많아요
- 720p 미만은 저장하지 않게 되어 있어서 정상입니다.
